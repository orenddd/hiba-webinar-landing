# היבא - צבעי המותג

## Brand Colors
| שם | Hex |
|---|---|
| Pink (ראשי) | `#f9226c` |
| Purple (ראשי) | `#9f21cf` |
| Pink-Purple | `#bb2eb6` |
| Light Pink | `#f1307b` |

## שימוש בצבעים
- כל פרויקט חדש עבור היבא חייב להשתמש בצבעים האלה בלבד
- אסור להשתמש בגוונים שונים (כמו `#fa226b` במקום `#f9226c`)
- גרדיאנט ראשי: `linear-gradient(135deg, #f9226c, #9f21cf)`
- גרדיאנט רחב: `linear-gradient(135deg, #f9226c, #bb2eb6, #9f21cf)`

## CSS Variables
```css
:root {
    --pink:        #f9226c;
    --purple:      #9f21cf;
    --pink-purple: #bb2eb6;
    --light-pink:  #f1307b;
}
```
